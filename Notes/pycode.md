# PYTHON CHEATSHEET


## Shortcut
### PyCharm
| Action | Keys |
| ---- | ---- |
| run |  |
| comment | `command` + `/` |
| uncomment | `command` + `/` |
| Auto alignment | `command` + `option` + `L` |

### Jupyter Notebook
| Action | Keys |
| ---- | ---- |
| run | `shift` + `enter` |

### VS code
| Action | Keys |
| ---- | ---- |
| run | `shift` + `enter` |
| comment | `command` + `/` |
| uncomment | `command` + `/` |
| format | `shift` + `option` + `F` |
